Ship Inspection Game: UI Design

Ship Inspection Game Design

I was a member of a team that developed a Ship Inspection Game for National Maritime Museum in Gdansk. I worked closely with students from the computer programming course to develop this game.

We started with the game idea and had to compose the interface to match the exhibition in the museum with colours, style and theme. To make the app accessible to more people, there was both Polish and English version of the whole game.

We made wireframes to create a working concept and prepared user flows to see if our design had all the elements needed. We then worked on various versions of the design and implemented it into the game before presenting to the museum staff.
